Get organized

Figma is a great tool for Prototyping UI/UX designs. It allows you to create interaction flows directly in the design file with advanced, yet easy-to-use features. In just a few minutes, you can turn your static designs into a realistic version of your website or product.

What's important

When you're ready to start Prototyping, add a copy of every screen and component needed to a new page. Arrange everything in a logical order with clear, identifiable names. Using copies will allow you to use the same designs in many Prototypes. While also keeping the originals safe and organized.
